# arduino_pomor

A [pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique) Timer for arduino.

# Features
* Shows current task progress
* Pause functionallity
* Beeper with increasing volume
* Real Time clock
* Sleep between actions to save battery
